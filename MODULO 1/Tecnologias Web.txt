HTML, CSS e Javascript

 

HTML - Hypertext Markup Language -  Linguagem utilizada para realizar a marcação de paginas web, é a base fundamental para qualquer site pois ela define os elementos como, cabeçalho, parágrafos, rodapés, listas, links, imagens , etc... o HTML oferece o esqueleto da pagina web, permitindo que o navegador interprete o código. 

CSS - Cascading Style Sheets - É uma linguagem de estilização de conteúdo, usando seletores por id ou classe o CSS permite que nossa pagina fique cada vez melhor para o usuário final, estilizando textos, imagens, margens, preenchimentos, cores e outros aspectos visuais, sem CSS tudo fica feio.

JAVASCRIPT - Linguagem de programação que permite criar interatividade e comportamento dinâmico a pagina web, permite criar funções para eventos, animações, validações, requisições assíncronas (AJAX), etc...

Ele permite que o dev deixe o site interativo, reativo e responsivo, melhorando a exp do usuário .

Unindo as três tecnologias temos o conjunto fundamental de linguagens para web, em suma é o core (nucelo) de qualquer ferramenta web, elas trabalham juntas deixando a pagina atraente, funcional e acessível, permitindo que o usuário navegue de maneira eficiente e interaja com o conteúdo apresentado.
